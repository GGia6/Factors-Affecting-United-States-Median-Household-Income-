# Factors Affecting Median Household Income in the United States

## Project Overview
This project investigates the relationship between race and different thresholds median household income, using a sample of 48 states from the United States. The variables included in analysis are: Social Security income, Public Assistance, population and population density, as well as the racial and gender proportions per state to determine the factors that surround median income.

For readability AI_AN pertains to the American Indian/ Alaskan Native racial group.
